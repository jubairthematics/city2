### Run Locally

To run **AH Couriers** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/codewithsadee/AH Couriers.git
```

Windows:

```bash
git clone https://github.com/codewithsadee/AH Couriers.git
```
